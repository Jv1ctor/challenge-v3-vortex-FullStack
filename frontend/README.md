# Desafio Vortex

Desafio de ingressão ao Vortex, este repositorio se trata da parte __web__ da aplicação. Desenvolvida com __Typescript + React + TailwindCss + Shadcn/ui__. Objetivo é desenvolver uma aplicação dashboard web para controle de um grupo industrial ficticio.

Essa aplicação receberá os registros de consumo de maquinas, enviados pelos operadores
através da aplicação mobile, a partir dos dados será exibido graficos indicativos. Outra responsábilidade será de adicionar, excluir e editar registros de leitura.
